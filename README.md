# Mototaxi Driver: 3D Simulation where you drive a mototaxi through the city

A little 3D videogame where you drive through Lima city in a mototaxi.

> [!IMPORTANT]
> There is NO boundaries, drive wherever you like.

## Technologies Used
- OpenStreetMap: To calculate the path from point A to point B
- Leaflet: To display the path in a map
- ThreeJS: To make the 3D simulations
